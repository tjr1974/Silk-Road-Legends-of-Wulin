Help System

Help

The help command displays useful information about specific topics.

Usage:
    help [topic]
Example:
    help navigation - This will display information about navigation commands.
    help commands - This will display information about common commands.
    help drop - This will display information about the drop command.
    help get - This will display information about the get command.
    help put - This will display information about the put command.
    help save - This will display information about the save command.

For additional information see: TOPICS, COMMANDS, NAVIGATION

Navigation

The game engine allows for travel in six directions: North, East, West, South, Up, and Down.

Usage:
    n
    e
    w
    s
    u
    d
Example:
    n - This moves the player character north of the current location.
    e - This moves the player character east of the current location.
    w - This moves the player character west of the current location.
    s - This moves the player character south of the current location.
    u - This moves the player character above the current location.
    d - This moves the player character below the current location.

For additional information see: LOOK, OPEN, CLOSE, LOCK, UNLOCK

Killer

This command toggles killer mode on or off. If killer mode is on, character will attempt to kill whatever he attacks. If killer mode is off, Character will refrain from killing and end combat when his opponent is unconscious.

Usage:
    killer
Example:
    killer

Aliases: KILLER
